# IDR — Institute of Digital Risk

A responsive single-page website for the **Institute of Digital Risk (IDR)**, built as a web design assignment using pure HTML, CSS, and vanilla JavaScript.

---

## Project Structure

```
idr-website-assignment/
├── index.html   — Single-page HTML with all sections
├── style.css    — Full styling (responsive, animated)
├── script.js    — Interactive behaviour
└── README.md    — This file
```

---

## Sections

| # | Section | Description |
|---|---------|-------------|
| 1 | **Hero** | Animated hero with title, subtitle, CTA, and stats bar |
| 2 | **About** | Mission, values, and animated orbit diagram |
| 3 | **Services** | Academy, Innovation, and Advisory cards |
| 4 | **Community** | Community features and benefits grid |
| 5 | **Contact** | Validated contact form with simulated submission |

---

## Design

- **Palette:** `#0a0a0a` (black) · `#ffffff` (white) · `#f97316` (orange)
- **Typography:** [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts
- **Style:** Cybersecurity / fintech — dark, minimal, and sharp

---

## Features

- Sticky navbar with scroll-triggered shadow
- Mobile hamburger menu (animated)
- Active nav link highlighting based on scroll position
- Smooth scrolling with offset for fixed navbar
- Scroll-reveal animations via IntersectionObserver
- Hero parallax on scroll
- Animated background grid, glowing orbs, rotating orbit rings
- Hover effects on service cards and community features
- Contact form with real-time validation and success feedback
- Fully responsive — mobile, tablet, desktop

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Markup | HTML5 (semantic) |
| Styling | Vanilla CSS (Flexbox + CSS Grid) |
| Scripting | Vanilla JavaScript (ES6+, IIFE) |
| Fonts | Google Fonts — Inter |

No frameworks. No libraries. No build tools required.

## How to Run

Open `index.html` in any modern web browser. No server or build step required.

```bash
# Or use a local static server
npx serve .
```
